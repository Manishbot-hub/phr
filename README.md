# Pharmacy Management WPF Application

This repository contains a **single-window** WPF application for pharmacy management with an attractive dashboard, keyboard-friendly navigation, and core workflows:

- Inventory management (add/update/delete/filter medicines)
- Sales and billing with tax calculation
- Supplier directory
- Prescription queue
- Dashboard insights (today sales, low stock, expiring soon)

## Keyboard Shortcuts

- `Ctrl + N`: New inventory item
- `Ctrl + S`: Save inventory item
- `Ctrl + Delete`: Delete selected item
- `Ctrl + F`: Focus the inventory form
- `F5`: Refresh dashboard numbers

## Run

```bash
dotnet restore
dotnet run
```

> Note: WPF requires Windows.
